a game for cpsc-25

CHANGES

    Modified Node so it can store Food
    Updated LinkedList so it works with the new Node
    Added the Ally class and used it in main

    We made the UI a bit cleaner with separators and added an option to exit game

EXPLANATION:

The program reads rooms from a CSV file and saves them in a LinkedList. Each node has a Room and a Food object. The player moves through the rooms by choosing options in the console.
